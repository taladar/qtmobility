Detailed Description

The QGalleryTypeRequest class provides an interface for requesting the properties of a type from a gallery.

QGalleryItemRequest executes a query which returns information summarizing items of the type specified in itemType. The query will return meta-data values which describe the type as a whole such as the total number of items of that type.

When the request has finished and if the type is one recognized by the gallery the valid property will be true, if not it will be false.

If the autoUpdate property is true when the request is executed it will enter an Idle state on finishing and will refresh the queried information if the type changes. If the gallery can't provide updates it will instead go immediately to the Finished state. Automatic updates can be canceled by calling cancel() on a idle request.

See also QDocumentGallery.

Property Documentation

autoUpdate : bool

This property holds whether a the results of a request should be updated after a request has finished.

If this is true the request will go into the Idle state when the request has finished rather than returning to Inactive.
