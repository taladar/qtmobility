QML Journal Element

The Journal element provides a journal which is associated with a particular point in time. More...

    Detailed Description

    The Journal element is part of the QtMobility.organizer 1.1 module.

    See also OrganizerItem, Event, EventOccurrence, Todo, TodoOccurrence, Note, and QOrganizerJournal.

    Property Documentation

    dateTime : date

    This property holds the date time associated with this journal.
