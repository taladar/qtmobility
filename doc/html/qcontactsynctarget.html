Detailed Description

The QContactSyncTarget class provides a sync target for a contact.

This leaf-class has been part of the default schema since version 1.0 of the Qt Mobility project.

Member Function Documentation

void QContactSyncTarget::setSyncTarget ( const QString & syncTarget )

Sets the identifier of the backend store to which the contact containing this detail should be synchronized to syncTarget.

See also syncTarget().

QString QContactSyncTarget::syncTarget () const

Returns the identifier of the backend store to which the contact containing this detail should be synchronized.
