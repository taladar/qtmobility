Grue Application

Files:

  • sensors/grueapp/main.cpp
  • sensors/grueapp/grueapp.pro

The Grue application example demonstrates the use of the Grue sensor which was defined and implemented by the Grue Plugin example. The Grue application is a commandline application. It is significant to note that there is no link-time dependency on the Grue plugin. The Grue application uses the generic access feature of the Sensors API.

See also Grue Plugin.
