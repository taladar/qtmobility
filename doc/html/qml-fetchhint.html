QML FetchHint Element

The FetchHint element provides hints to the manager about which organizer item information needs to be retrieved in an asynchronous fetch request or a synchronous function call. More...

  • List of all members, including inherited members

    • Properties

    • optimizationHints : FetchHint::OptimizationHints

    Detailed Description

    This element is part of the QtMobility.organizer 1.1 module.

    See also QOrganizerItemFetchHint.

    Property Documentation

    optimizationHints : FetchHint::OptimizationHints

    This property holds the optimization hint flags specified by the client. These hints may be ignored by the backend, in which case it will return the full set of information accessible in a organizer item.
