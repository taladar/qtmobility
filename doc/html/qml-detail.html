QML Detail Element

The Detail element represents a single, complete detail about a organizer item. More...

  • List of all members, including inherited members

    • Properties

    • definitionName : string
    • fieldNames : list<string>
    • readOnly : bool
    • removable : bool
    • type : enumeration

    Detailed Description

    The Detail element is part of the QtMobility.organizer 1.1 module.

    See also QOrganizerItemDetail.

    Property Documentation

    read-onlydefinitionName : string

    This property holds the string constant for the definition name of the detail. This property is read only.


    read-onlyfieldNames : list<string>

    This property holds the list of all field names which this detail supports.

    This property is read only.


    read-onlyreadOnly : bool

    This property indicates whether or not this detail is writable. This property is read only.


    read-onlyremovable : bool

    This property indicates whether or not this detail is removale. This property is read only.


    read-onlytype : enumeration

    This property holds the type of the detail and is read only. It can be one of:

    • Detail.EventTime
    • Detail.JournalTime
    • Detail.TodoTime
    • Detail.TodoProgress
    • Detail.AudibleReminder
    • Detail.VisualReminder
    • Detail.EmailReminder
    • Detail.Comment
    • Detail.Description
    • Detail.DisplayLabel
    • Detail.Guid
    • Detail.Location
    • Detail.Parent
    • Detail.Priority
    • Detail.Recurrence
    • Detail.Timestamp
    • Detail.Type
    • Detail.Tag
