Declarative Map Viewer Example

Files:

  • declarative-location/mapviewer/TitleBar.qml
  • declarative-location/mapviewer/mapviewer.qml
  • declarative-location/mapviewer/qmlmapviewer.cpp
  • declarative-location/mapviewer/mapviewer.pro
  • declarative-location/mapviewer/mapviewer.qrc

This example demonstrates the basic use of the QML Map element.

Moving the mouse while holding down the left mouse button pans the map. Double-clicking the mouse centers and zooms in on the point that was clicked. The '+' and '-' keys can be used to change the zoom level.
