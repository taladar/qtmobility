All Namespaces

This is a list of the main namespaces in the QtMobility API.

QBluetooth

Contains functions and definitions related to Bluetooth

QMessageDataComparator

Contains types used in specifying the comparison of MessageStore objects with user-defined values

QService

Contains miscellaneous identifiers used throughout the Qt Service framework library

QValueSpace

Contains miscellaneous identifiers used throughtout the Publish and Subscribe API

QtMultimediaKit

Contains miscellaneous identifiers used throughout the Qt Media services library
