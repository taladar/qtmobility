    Detailed Description

    The LandmarkProximityFilter element specifies a proximity filter for landmark model. Logical combinations of this and other landmark filters can be formed using LandmarkIntersectionFilter and LandmarkUnionFilter.

    This element is part of the QtMobility.location 1.1 module.

    The following example creates a filter which filters for landmarks located within 1500 metres of the current position.

         PositionSource {
         id: myPosition
         updateInterval: 1000
     }
     LandmarkProximityFilter {
         id: landmarkFilterMyCurrentLocation
         center: myPosition.position.coordinate
         radius: 1500
     }

    See also LandmarkModel and QLandmarkProximityFilter.

    Property Documentation

    center : Coordinate

    Center coordinate in whose proximity landmarks should be filtered.

    See also radius.


    radius : double

    Radius (in metres) from the center coordinate to filter (landmarks within that area).

    See also center.
