Detailed Description

The QOrganizerCollectionFetchRequest class allows a client to asynchronously request collections from an organizer manager.

For a QOrganizerCollectionFetchRequest, the resultsAvailable() signal will be emitted when the resultant collections (which may be retrieved by calling collections()), are updated, as well as if the overall operation error (which may be retrieved by calling error()) is updated.
